## 📝 Заметки

### Важные технические детали:
- Python 3.14: `asyncio.get_event_loop()` устарел
- VK Long Poll требует положительный ID группы
- Inline клавиатуры: макс 10 строк
- Callback payload в VK уже dict (не строка)

### Файлы для редактирования:
- `main.py` — основная логика и обработчики
- `database.py` — функции для работы с БД
- `config.py` — конфигурация

### Тестирование:
- Локально: `python main.py`
- На сервере: `vkactl logs`
