[![vk-api](https://img.shields.io/badge/vk--api-2.x-green.svg)](https://vk-api.readthedocs.io/)

## ✨ Возможности

### 🛒 Список покупок
- **Автоматическая классификация** товаров по магазинам:
  - **Магнит** — продукты питания
  - **Фикспрайс** — бытовые товары
  - **Другое** — всё остальное
- **Ручное распределение** с помощью префиксов
- **Отметка купленных товаров**
- **Пакетное добавление** через запятую

### 📋 Список дел
- Добавление задач с **датами выполнения**
- **Напоминания** о делах (ежедневно в 9:00)
- Распознавание дат: `завтра`, `15.03`, `15 марта`, `на 15.03`

### 📚 Учёба | 💡 Идеи | 🍳 Рецепты

### 🌤 Погода
- **Ежедневные прогнозы** в заданное время
- **Уведомления о дожде**
- Настройка города и времени

### ⚙️ Настройки
- Названия магазинов и сокращения
- Триггерные слова
- Видимость кнопок меню
- Настройки погоды

## 🚀 Быстрая установка

### Linux сервер (рекомендуется):

```bash
curl -sSL https://raw.githubusercontent.com/Drentis/VKAssistant/main/install.sh | sudo bash
```

Или напрямую:

```bash
curl -sSL https://raw.githubusercontent.com/Drentis/VKAssistant/main/deploy.sh | sudo bash
```

### Windows:

```powershell
cd $HOME
git clone https://github.com/Drentis/VKAssistant.git
cd VKAssistant
.\install.ps1
```

## ⚙️ Настройка

### 1. Создание сообщества ВКонтакте

1. Перейдите на vk.com и создайте сообщество (тип "Бот")
2. **Управление сообществом** → **Работа с API**
3. Создайте ключ с правами: `messages`, `manage`
4. Скопируйте токен

### 2. Получение ID сообщества

ID виден в URL: `vk.com/club123456789` → ID: `-123456789`

### 3. Настройка .env

```env
VK_GROUP_ID=-123456789
VK_TOKEN=your_vk_token
WEATHER_API_KEY=your_weather_key
ADMIN_ID=your_vk_id
```

## 🚀 Запуск

### На VPS (systemd)

После установки бот работает как сервис:

```bash
vkactl status    # Статус
vkactl logs      # Логи
vkactl restart   # Перезапуск
vkactl update    # Обновление
vkactl edit      # Настройки
vkactl delete    # Удаление
vkactl version   # Версия
```

### Локально

```bash
python main.py
```

| `vkactl delete` | Удалить бота |
| `vkactl version` | Показать версию |

## 🐛 Устранение неполадок

### Бот не отвечает
1. `vkactl status` — проверьте статус
2. `vkactl logs` — проверьте логи
3. Проверьте токен в `.env`

### Не работает погода
1. Проверьте API ключ
2. Убедитесь что город установлен

### Напоминания не приходят
1. Бот должен работать постоянно
2. Задачи должны иметь дату
