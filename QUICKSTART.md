# 🚀 Быстрый старт VKAssistant

## За 5 минут к работающему боту

### Шаг 1: Установка (1 минута)

**Windows:**
```powershell
git clone https://github.com/Drentis/VKAssistant.git
cd VKAssistant
.\install.ps1
```

**Linux:**
```bash
git clone https://github.com/Drentis/VKAssistant.git
cd VKAssistant
chmod +x install.sh
./install.sh
```

### Шаг 2: Создание бота ВКонтакте (2 минуты)

1. **Создайте сообщество:**
   - Перейдите на vk.com/groups?act=create
   - Выберите тип "Бот"
   - Дайте название (например, "Мой Помощник")

2. **Получите токен:**
   - Перейдите в Управление сообществом → Работа с API
   - Создайте ключ доступа
   - Скопируйте токен

3. **Узнайте ID сообщества:**
   - Перейдите в ваше сообщество
   - ID виден в URL: vk.com/club**123456789** → ID: `-123456789`
   - **Важно:** Добавьте минус в начало!

4. **Узнайте ваш VK ID:**
   - Перейдите на вашу страницу
   - ID виден в URL: vk.com/id**123456789** → ID: `123456789`

### Шаг 3: Настройка .env (1 минута)

Откройте файл `.env` и заполните:

```env
VK_GROUP_ID=-123456789          # ID сообщества (с минусом!)
VK_TOKEN=abcdef1234567890...    # Токен из управления сообществом
WEATHER_API_KEY=                # Опционально, для погоды
ADMIN_ID=123456789              # Ваш VK ID
```

### Шаг 4: Запуск (10 секунд)

**Windows:**
```powershell
.\venv\Scripts\python.exe main.py
```

**Linux:**
```bash
./venv/bin/python main.py
```

### Шаг 5: Проверка (30 секунд)

1. Перейдите в ваше сообщество
2. Напишите боту: `/start`
3. Вы должны увидеть приветственное сообщение с меню

## 🆘 Проблемы?

### "Бот не отвечает"
- Проверьте, что Long Poll включён в настройках сообщества
- Убедитесь, что токен правильный
- Проверьте, что бот добавлен в сообщество

### "Ошибка авторизации"
- Проверьте VK_TOKEN в .env
- Убедитесь, что у токена есть права на сообщения

### "Неверный ID сообщества"
- ID должен быть отрицательным (с минусом): `-123456789`
- Проверьте, что это именно ID сообщества, а не группы
