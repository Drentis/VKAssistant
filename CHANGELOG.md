## [1.0.1] - 2026-04-05

### 🔧 Исправления

**Пагинация списков редактирования:**
- **Проблема:** VK inline клавиатуры поддерживают макс **4 строки** (обычные — 10). При 16+ элементах бот падал с ошибкой `[911] Keyboard format is invalid: buttons contain too much rows`
- **Решение:** Добавлена пагинация для всех списов (🛒 Покупки, 📋 Дела, 📚 Учёба, 💡 Идеи)

**Технические детали:**
- `get_edit_keyboard()`: `items_per_page` — shopping=4, todo/study/ideas=2
- Compact layout: shopping — по 2 элемента в строке; остальные — 1 элемент = 1 строка (✏️ текст | 🗑)
- Навигация: `⬅️ prev/next ➡️` кнопки в payload с `{"type": "edit_list", ..., "page": N}`
- Page сохраняется в payload каждой кнопки (`toggle_item`, `delete_item`, `edit_item`, `clear_list`)
- Жёсткий лимит `buttons[:4]` перед отправкой
- `handle_item_edit_input()`: после редактирования возвращается на ту же страницу

**Обновлённые обработчики:**
- `edit_list` — читает `page` из payload, показывает элементы текущей страницы
- `toggle_item` — сохраняет страницу при обновлении статуса
- `delete_item` — возвращается на ту же страницу после удаления
- `edit_item` — сохраняет `page` и `category` в state
- `clear_list` — кнопка "Назад" через inline (не main keyboard)

## [1.0.0] - 2026-04-04

### 🎉 Первый релиз

VKAssistant — бот для ведения списков и заметок ВКонтакте.

### ✨ Возможности

**🛒 Список покупок:**
- Автоматическая классификация товаров по магазинам (Магнит/Фикспрайс/Другое)
- Ручной выбор магазина через префиксы (м купить хлеб, ф купить шампунь)
- Inline кнопки для редактирования (отметка купленного, удаление, очистка)
- Редактирование элементов todo/study/ideas

**📋 Список дел:**
- Добавление задач с датами (завтра, 15.03, 12 марта)
- Ежедневные напоминания в 9:00 о делах на завтра
- Inline кнопки для управления

**📚 Учёба:**
- Список учебных задач
- Inline кнопки для редактирования

**💡 Идеи:**
- Быстрые заметки
- Inline кнопки для редактирования

**🍳 Рецепты:**
- Добавление рецептов с ингредиентами
- FSM для пошагового добавления (название → ингредиенты → описание)
- Кнопка «Добавить в корзину» — перенос ингредиентов в покупки
- Inline кнопки для управления

**🌤 Погода:**
- Прогноз погоды по запросу
- Ежедневная отправка прогноза в настроенное время
- Уведомления о дожде
- Учёт часового пояса города через OpenWeatherMap API

**⚙️ Настройки:**
- Настройка названий магазинов (название, описание, сокращение)
- Изменение триггерных слов (покупки, дела, учёба, идеи, рецепты)
- Управление видимостью кнопок меню
- Настройка погоды (город, время, ежедневный прогноз, уведомления о дожде)
- Сброс профиля

**📂 Пользовательские категории:**
- Создание своих магазинов/категорий
- Настройка ключевых слов для автоклассификации
- Inline кнопки для просмотра, редактирования, удаления
- Команда `/categories`

**👤 Админ-панель:**
- Общая статистика (пользователи, товары, задачи, идеи, рецепты)
- Личная статистика
- Очистка данных пользователя
- Обновление бота через Git (`/update`)

### 🔧 Технические детали

- **Python 3.14** — поддержка актуальной версии
- **VK Long Poll** — стабильное подключение
- **aiosqlite** — асинхронная работа с SQLite
- **Оптимизация** — глобальный event loop для запросов к БД (ускорение ~5x)
- **Установка** — Linux (deploy.sh + vkactl), Windows (install.ps1)

### 📦 Зависимости

- `vk-api>=2.0.0` — VK API + Long Poll
- `aiosqlite>=0.19.0` — асинхронная БД
- `aiohttp>=3.8.0` — HTTP запросы для погоды
- `python-dotenv>=1.0.0` — загрузка .env
